## Project Name => DigiTools – Premium Digital Tools Platform

## 📝 Description => DigiTools is a modern and responsive web application built using React. It provides a collection of premium digital tools such as design templates, AI writing tools, and stock assets. Users can بسهولة browse products, add them to a cart, and proceed to checkout with real-time updates and notifications.

## ⚙️ Technologies Used
# React.js
# Tailwind CSS
# DaisyUI
# JavaScript (ES6+)
# React-Toastify


### ✨ Key Features

 1. Cart Functionality

Users can add products to the cart, remove items, and complete checkout. The cart updates dynamically and shows total selected items.

 2. Product & Cart Toggle System

A smooth toggle system allows users to switch between product listing and cart view instantly.

 3. Toast Notifications

Interactive alerts using React-Toastify for actions like adding to cart, removing items, and checkout confirmation.
